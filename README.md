# 순공부시간 기록 (Study Tracker)

버튼 한 번으로 **공부 시작**, 다시 누르면 **공부 종료 + 순공부시간 저장**하는 초간단 웹앱입니다.  
아이폰/아이패드에서도 쓰기 편하게 **GitHub Pages**에 올려두고 사용하도록 만들었습니다.

> 데이터는 브라우저의 로컬 저장소(localStorage)에 저장됩니다.  
> 안전을 위해 주기적으로 **내보내기(JSON)**로 백업하는 것을 권장합니다.

---

## 기능

- ✅ **시작/종료 버튼**으로 세션 기록
- ✅ 앱이 꺼져도 **시작 상태 유지**  
  (시작 후 앱을 닫아도 시작 시간은 저장되고, 다시 열어서 종료를 누르면 그 사이 시간이 기록됩니다)
- ✅ **월 달력**에서 날짜별 누적 공부시간 표시
- ✅ 월 평균 공부시간 표시
- ✅ 날짜 클릭 시 **해당 날짜 세션 목록** 확인
- ✅ 세션 **개별 삭제(🗑)** 지원
- ✅ **자정 넘는 세션 자동 분할** (날짜별 누적 정확하게)
- ✅ **백업 리마인더**: 마지막 내보내기 날짜 표시 + 7일 이상 경고

---

## 사용 방법

### 1) 웹에서 사용
GitHub Pages 주소로 접속해서 사용합니다.

### 2) 아이폰 홈 화면에 추가(추천)
1. 사파리에서 사이트 열기
2. 공유 버튼(⬆️) → **홈 화면에 추가**
3. 아이콘처럼 실행 가능

---

## 백업 / 복구

- **내보내기(JSON)**: 현재 기록을 파일로 저장합니다. (이 동작이 “백업 완료”로 기록됨)
- **가져오기(JSON)**: 저장해 둔 백업 파일로 복구합니다.

> iOS 환경에서는 저장소가 예기치 않게 정리될 수 있으니  
> **주 1회 정도 백업**을 권장합니다.

---

## 개발 메모

- 단일 `index.html` 파일로 동작합니다.
- 저장 데이터 형식(예시):
  - `sessions`: `{ id, start, end, ms }[]`
  - `runningStart`: 진행 중인 경우 시작 시각(ms)
  - `ms`는 밀리초 단위로 저장되며, 화면에는 기본적으로 “시:분” 형태로 표시됩니다.
